## v1.6.2

### 🚀 Features
- **每排雨滴宽度**：新增 `RainWidthRow1/2/3` 独立滑块（10–200px），每排雨滴可单独设置宽度
- **每排雨滴速度/高度上限提升**：slider 从 1000 扩展到 2000，text field 上限完全放开
- **韩语支持**：新增 `ko` 语言选项

### 🐛 Bug Fixes
- **修复首次按键雨滴不显示**：`RainSystem.PreWarmPool` 在无有效 Canvas 时预创建对象导致 CanvasRenderer/mesh 未正确初始化，移除预创建后 Rain 对象轻量无感知，从根本上修复
- **Color foldout 每帧重置子项状态**：`DrawColorSection` foldout 展开状态每帧被重置，导致子项无法保持展开/折叠
- **KPS/Total sub-foldout 判断**：用 `>=0` 代替 `==t` 导致折叠/展开状态判断错误

### 🔧 Performance
- **零分配 KPS/Count 显示**：`NumBuffer.Format` 预分配 char[32] + `TMP_Text.SetText(buf, offset, length)`，彻底消除每帧 `ToString` 的字符串分配
- **Queue 预分配**：`PressTimes` (256) 和 per-key KPS 队列 (32) 预分配容量，消除游戏中的 Queue 扩容分配
- **Rain 对象池启动预热**：10 Rain + 20 RawRain 启动时预创建（但后续移除了预热修复 Bug，已回退）
- **Idle skip**：`_hasKeyPressActivity` 标志位，无按键时跳过 `ProcessPerKeyKpsInUpdate` 循环

### 🧹 Refactor
- **Profile 系统**：`KeyViewerSettings` 拆分为 `ProfileData`（全部配置）+ 元数据壳（Version/CurrentProfile/ProfileNames/Language），支持多 Profile 创建/切换/重命名/删除，v2→v3 自动迁移
- **GUI 重构**：提取 `FloatSliderField` / `DrawFoldoutButton` / `DrawFoldoutItemButton` 消除重复；`DrawSettingsWindow` 拆分为 10+ 独立 section 方法
- **RainSystem 重构**：`UpdateEffects` 拆分为 `SyncCachedSpeeds` / `UpdateSingleRainDrop` / `ApplyRainTransforms` / `UpdateFadeOut` / `UpdateTrailEdge`；统一雨滴颜色/鬼键颜色获取
- **AddQuad 参数简化**：7 参数 → 4 参数 `(VertexHelper, Rect, Color, Color)`
- **I18n 重写**：手写 JSON 解析器替换为 `JsonUtility.FromJson<LangFile>`
- **FileBased 字体样式**：补齐 `FontStyleFlags` 支持（Bold/Italic 等）
